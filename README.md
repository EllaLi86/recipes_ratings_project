# Investigation on the Relationship Between Cooking Time and Recipe Scores
### Authors: Xinyuan(Ella) Li
## Overview
This data science project, conducted at UCSD, focus on exploring the relationship between the cooking time and rating of a recipe of the given recipe.

## Introduction
This project analyzes two interconnected datasets containing recipe information and user interactions from a popular recipe sharing platform.
	**Raw Recipes Dataset**
The first dataset contains 83,782 recipes with 12 columns of detailed recipe information. Each row represents a unique recipe with attributes including preparation time, ingredients, cooking steps, and contributor information. The dataset has 83,782 unique recipe IDs, confirming that each row represents a distinct recipe.

	**Interactions Dataset**
The second dataset contains 731,927 user interactions with 5 columns showing how users have engaged with these recipes. Each row represents a single user rating and review for a specific recipe, allowing for multiple ratings per recipe from different users.
